# Xyntriq ARAT MVP

AI Readiness Assessment Tool (ARAT) for Xyntriq.

## Technology Stack
- **Framework**: Next.js 15 (App Router)
- **Database**: SQLite (via Prisma)
- **Styling**: Tailwind CSS
- **PDF**: @react-pdf/renderer

## Getting Started

1. `npm install`
2. `npx prisma db push`
3. `npm run dev`

## Key Locations
- **Rules**: `src/lib/rules/scoring-rules.json`
- **Schema**: `prisma/schema.prisma`
- **Admin**: `/admin`
